# 后台管理说明

仅**超级管理员**可进入后台管理页面

包含以下管理页面：

- **用户管理**

  - 可**查看**、**编辑**、**创建**平台用户。

  - 可配置用户的**登录密码**、**用户级别**、**超级管理员**等。

  ![用户管理](../images/manage_01.png)

- **分析记录管理**

  - 可查看平台**全部分析记录**。

  - 可点击查阅**分析记录详情**。

  ![分析记录管理](../images/manage_02.png)

- **节点管理**

  - 可查看**常驻节点状态**。

  - 可**查看**、**编辑**、**删除**常驻节点。

  - 可配置节点**工具进程**。

  - 可配置**标签**

  ![节点管理](../images/manage_03.png)
